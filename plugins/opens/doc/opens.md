# Collecting OPENs/FIXMEs

The `opens` plugin allows to create lists of OPENs/FIXMEs/TODOs in the document. Here's the example
of how it could be used:

    <!--
    # Items to close

    ```opens("OPEN")
    ```

    # Required fixes

    ```opens("FIXME", numbered=True)
    ```

    ```opens("TODO", numbered=True)
    ```
    -->
